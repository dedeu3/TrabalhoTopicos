PROPOSTA INICIAL DE PROJETO

Propomos como projeto para a disciplina de Tópicos em Computação do semestre 01/2017, a elaboração de uma aplicação web open source que funcione como Gerador de Senhas. Deverá funcionar da seguinte maneira: o usuário faz sua autenticação e preenche alguns campos pré-definidos, que funcionarão como palavras-chave para geração das senhas (por exemplo: hobbies, qualidades, restaurantes preferidos, destinos de viagens, datas importantes, etc.). Com base nos dados fornecidos, a aplicação gerará senhas aleatórias ao usuário, que poderão ser anotadas e utilizadas em outros locais, conforme lhe for viável.
Sendo assim, a aplicação deverá conter:
Tela para cadastro de novos usuários;
Recuperação de senhas (campo esqueci minha senha);
Tela de preenchimento das palavras-chave;
Geração das senhas aleatórias.
